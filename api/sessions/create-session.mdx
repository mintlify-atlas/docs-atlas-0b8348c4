---
title: "Create Session"
api: "POST /sessions"
---

Create a new Wormkey session with a unique slug and tokens for owner access.

## Request Body

<ParamField body="port" type="number" default="3000">
  The local port number your application is running on
</ParamField>

<ParamField body="authMode" type="string" default="none">
  Authentication mode for the session. Options: `none`, `basic`
  
  When set to `basic`, the response will include generated `username` and `password` credentials.
</ParamField>

<ParamField body="expiresIn" type="string" default="24h">
  Session expiration duration. Accepts formats like `30m` (minutes) or `24h` (hours)
</ParamField>

## Response

<ResponseField name="sessionId" type="string" required>
  Unique session identifier (format: `sess_<token>`)
</ResponseField>

<ResponseField name="slug" type="string" required>
  Human-readable session slug (format: `adjective-noun-number`)
</ResponseField>

<ResponseField name="publicUrl" type="string" required>
  Public URL where viewers can access the session
</ResponseField>

<ResponseField name="ownerUrl" type="string" required>
  Owner dashboard URL with authentication token included
</ResponseField>

<ResponseField name="overlayScriptUrl" type="string" required>
  URL to the overlay script for embedding session controls
</ResponseField>

<ResponseField name="edgeUrl" type="string" required>
  WebSocket edge URL for tunnel connection
</ResponseField>

<ResponseField name="sessionToken" type="string" required>
  Combined token for session authentication (format: `slug.ownerToken`)
</ResponseField>

<ResponseField name="expiresAt" type="string" required>
  ISO 8601 timestamp when the session expires
</ResponseField>

<ResponseField name="username" type="string">
  Basic auth username (only present when `authMode` is `basic`)
</ResponseField>

<ResponseField name="password" type="string">
  Basic auth password (only present when `authMode` is `basic`)
</ResponseField>

## Example Request

```bash
curl -X POST https://control.wormkey.io/sessions \
  -H "Content-Type: application/json" \
  -d '{
    "port": 3000,
    "authMode": "none",
    "expiresIn": "24h"
  }'
```
