---
title: "Update Session Policy"
api: "POST /sessions/by-slug/:slug/policy"
---

Update the access policy for a session, including visibility, viewer limits, blocked paths, and password protection.

## Path Parameters

<ParamField path="slug" type="string" required>
  The session slug (format: `adjective-noun-number`)
</ParamField>

## Request Body

All fields are optional. Only the fields provided will be updated.

<ParamField body="public" type="boolean">
  Whether the session is publicly accessible
  
  When set to `false`, the session becomes private and requires additional authentication.
</ParamField>

<ParamField body="maxConcurrentViewers" type="number">
  Maximum number of concurrent viewers allowed in the session
  
  Default is 20. Set to a lower number to limit concurrent access.
</ParamField>

<ParamField body="blockPaths" type="string[]">
  Array of URL paths to block from being accessed by viewers
  
  Example: `["/admin", "/api/internal"]`
</ParamField>

<ParamField body="password" type="string">
  Password required for viewers to access the session
  
  Set to an empty string to remove password protection.
</ParamField>

## Response

<ResponseField name="ok" type="boolean" required>
  Whether the update was successful
</ResponseField>

<ResponseField name="policy" type="object" required>
  The updated policy object
  
  <ResponseField name="public" type="boolean" required>
    Whether the session is publicly accessible
  </ResponseField>
  
  <ResponseField name="maxConcurrentViewers" type="number" required>
    Maximum number of concurrent viewers allowed
  </ResponseField>
  
  <ResponseField name="blockPaths" type="string[]" required>
    Array of blocked URL paths
  </ResponseField>
  
  <ResponseField name="password" type="string" required>
    Viewer access password
  </ResponseField>
</ResponseField>

## Example Request

```bash
curl -X POST https://control.wormkey.io/sessions/by-slug/quiet-lime-42/policy \
  -H "Content-Type: application/json" \
  -d '{
    "public": false,
    "maxConcurrentViewers": 5,
    "blockPaths": ["/admin", "/api/internal"],
    "password": "secure123"
  }'
```

## Example Response

```json
{
  "ok": true,
  "policy": {
    "public": false,
    "maxConcurrentViewers": 5,
    "blockPaths": ["/admin", "/api/internal"],
    "password": "secure123"
  }
}
```

## Partial Update Example

You can update individual fields without affecting others:

```bash
curl -X POST https://control.wormkey.io/sessions/by-slug/quiet-lime-42/policy \
  -H "Content-Type: application/json" \
  -d '{
    "maxConcurrentViewers": 10
  }'
```

```json
{
  "ok": true,
  "policy": {
    "public": true,
    "maxConcurrentViewers": 10,
    "blockPaths": [],
    "password": ""
  }
}
```

## Error Response

Returns a 404 error if the session is not found:

```json
{
  "error": "Session not found"
}
```
